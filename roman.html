A subpar Roman-based googological notation

Trying to make Roman numerals into a googology thing by shamelessly ripping off of sys128's ideas.

Click me if you don't know what Roman numerals are

Background

Roman numerals are a non-positional number system that was used in Rome and is sometimes still used today.

Base symbols

Various multiples of 1/10 each have their own symbols, which are each combined in a certain way to represent larger numbers.
Symbol  Value
I 1
V 5
X 10
L 50
C 100
D 500
M 1,000

Strings of symbols add the values, but you shouldn't have more than 3 in a row. A power-of-ten symbol which appears before any of the two symbols after it on the chart (I before V or X, X before L or C, or C before D or M) subtracts. This could be modified so that multiple symbols subtract more (e.g. IIX = 8), or so that any symbol can subtract from any symbol (e.g. IC = 99), but the extension detailed in this page doesn't do that.

With thanks to sys128

A bar over a symbol indicates that its value is multiplied by 1,000. This is still a vanilla feature; L (Unicode "L̅") is 50,000. sys128 had the great idea to add more lines — as a side note, making more than 2 lines with CSS is an absolute nightmare.

V
(Unicode "V̅̅̅") = 5,000,000,000. Overlined numerals can be used next to non-overlined ones; XXMMMCCCIV = 23,304.
